  • cross_sectional_imputation
    The function works on imputation for the given cross sectional dataframe. For columns with numeric values, the null values of that column are imputed/filled with the mean (average) value of the column. For columns with string values, the null values of that column are imputed/filled with the mode (most frequent) value of that column.
    def cross_sectional_imputation(cross_sectional_df: pd.DataFrame, target_name: str) -> pd.DataFrame:
    Paramters:
    > cross_sectional_df (pd.DataFrame) : The dataframe of the cross-sectional dataset with the target column included.
    > target_name (str) : The name of the target column.
    Return:
    Optional[pd.DataFrame]: It will return the modified dataframe if the function is successful.
  • time_series_imputation

  • feature_scaling
    The function scales continuous-numerical values of the dataset. If the values in the column have a normal distribution then we do StandardScaling. If it does not, then we do MinMaxScaling Note : If the unique value count of the column is greater than 3, only then will the function apply scaling to that column.
    def feature_scaling(features: pd.DataFrame, unscale_columns: List[str]) -> pd.DataFrame:
    Paramters:
    > features (str) : The dataframe containing the features to be scaled.
    > unscale_columns (str) : The list of column names that represent the columns that should not be scaled.
    Return:
    pd.DataFrame: The dataframe consisting of all the scaled features.

    Return:
    pd.DataFrame: The proccessed dataframe.
